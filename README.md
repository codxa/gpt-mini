# gpt-mini
Merhabalar, openai dan api key alıp kendimi geliştirmek için yaptığım gpt-mini. yapmamdaki temel amaç bir siber güvenlik asistanı yapmaktı lazım olunca kod parçalarını vermesi, kod analizi yapması ve pentest konsunda yardım etmesi. Bunu uzun zaman önce yapmıştım geliştirmeyi bıraktım. devam edersem buradan yeni versiyonunu paylaşırım.

PyCharm Community Edition kullandım, gpt-mini özellikleri:

1-kod analizi yapmak

2-attığınız kodu analiz edip değiştirmek

3-sohbet geçmişi

4-dosya yükleme 

5-hazır kod parçaları sunmak

6-buna benzer özelikler...
